# Cypress e2e Baseline Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a local TypeScript Cypress e2e baseline in `tanstack-app` (config, scripts, home smoke test, README) while keeping Vitest for unit/component tests.

**Architecture:** Install Cypress + `start-server-and-test` under `tanstack-app`. Configure e2e-only `cypress.config.ts` with `baseUrl: http://localhost:3000`. Add one smoke spec that visits `/` and asserts the Home heading. Wire `cy:open`, `cy:run`, and `test:e2e` (auto-start `pnpm dev`). Document both test layers in the README.

**Tech Stack:** Cypress (latest), start-server-and-test, TypeScript, pnpm, TanStack Start (Vite, port 3000), Vitest (unchanged)

**Spec:** `docs/superpowers/specs/2026-07-19-cypress-e2e-design.md`
